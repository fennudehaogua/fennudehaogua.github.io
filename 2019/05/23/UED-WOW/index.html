UED&WOW

发表于 2019-05-23

UED

论文：Uniform Embedding for Efficient JPEG Steganography(2014,TIFS,CR18)

设计思想：

本篇论文指出DCT系数在嵌入修改时，若任意选择嵌入位置，由于小值JPEG系数的数量较多，则被选择到小值的可能性更大，这样，小值系数上的分布会变化更剧烈，更可能造成系数分布的形变，从而会引起较大的抗检测隐患，提出针对这一点的均衡失真隐写算法UED(Uniform Embedding Distortion)。

而如果每个值上的变化接近，则更可能保持JPEG系数的分布。因此，UED的设计目标是，压制在更多出现的小值系数上修改，优先选择更少出现的大值系数修改。

UED的修改系数选择原则：优先选择大值系数修改。图中，a是JPEG量化系数分布，b是希望的选择概率，c是所选择系数的分布（更均匀）

基于DCT系数本身及其块内、块间相邻系数的取值构造嵌入失真函数，并使用STC编码嵌入，减少了失真并提高了嵌入的效率的UED-SC( Uniform Embedding Distortion based on Single Coefficient)、UED-JC(UED based on Joint Coefficients)和SI-UED(Side Informed UED)。

基于以上失真函数，UED采用STC优化嵌入，不使用系数为0的样点，当需要修改且𝑐𝑖𝑗绝对值大于1时，采用随机±1的方式；当需要修改且𝑐𝑖𝑗绝对值为1时，将其改为𝑐𝑖𝑗+sgn(𝑐𝑖𝑗)，即+1改为2，−1改为−2。（sgn(0)=0,sgn(负数)=-1,sgn(正数)=1）。这样我们就不会增加额外的0值DCT系数。从而提高嵌入的效率。

提出的隐写机制：

用CC-JRM与CC-PEV特征进行相应的隐写分析，结果显示，UED显著优于非自适应的方法，但是，在主要攻击下它的安全性低于J-UNIWARD（不过UED速度显著更快）

UED算法引入了均匀嵌入思想，以相同的概率修改不同大小的非零量化DCT系数（原本选取小的DCT系数值概率大，大的DCT系数值概率小），在度量失真代价时综合考虑块内和块间邻域DCT系数的取值情况，使图像一阶和二阶统计特性的改变达到最小。

UERD是UED的改进算法，它将所有DCT系数(包括直流和零值DCT系数)均视为可嵌入的载体元素，并考虑了图像模型统计特性的变化，使嵌入修改与系数差异成正比，能够抵抗高维的隐写检测特征。用与DCT块对应的像素块的灰度共生矩阵来计算分块的纹理复杂程度，并结合均匀嵌入思想，将秘密信息嵌入分散到取值不同的DCT系数上，增加了隐写的自适应性。

WOW

论文：Designing steganographic distortion using directional filters(2012,WIFS,CR106)

设计思想：

WOW隐写算法的核心思想主要是利用空间域图像每个点在多个方向上的方向残差来衡量每个点的嵌入代价，利用这种方法可以使图像中在每个方向都难以建模的那些像素区域具有小的嵌入成本，而图像平滑区域和边缘区域会具有较大的嵌入成本。具体的过程可以分成三个步骤:首先，利用三个不同的方向滤波器获得载体图像每个像素点的方向残差，并利用这些方向残差计算得到每个像素的嵌入适应性;然后，利用这些嵌入适应性进行聚合得到每个像素点的嵌入代价;最后，在建立完以上的失真函数得到每个像素点的嵌入代价以后，利用STC编码结合这些嵌入代价实现最小化失真隐写并得到最终嵌入信息的图像。

小波系数反映了信号局部区域的时频特性，失真函数的设计可以以隐写后这种特性得到更多保持为原则。WOW是空间域图像自适应隐写，它在小波变换的一级分解LH、HL与HH子带上对系数改动量与原系数做相关运算，总体相关性越大则失真越低。

WOW核心思想是选择了方向性Daubechies 8小波，直接用小波从多个方向分解获得载体图像残差，进而构造失真函数。低通、高通滤波器𝒉，𝒈进行小波滤波获得一级分解LH、HL与HH三个子带系数的二维滤波器系数矩阵是：𝑲(1)= 𝒉 ∙ 𝒈T, 𝑲 2 = 𝒈 ∙ 𝒉T, 𝑲(3) = 𝒈∙𝒈T。它们滤波后得到的三个子带系数也称为三组残差（Residual），分别为𝑹(𝑘) = 𝑲 𝑘∗𝑿, 𝑘 =1,2,3，“∗”表示二维卷积。然后对残差值进行加权、聚合，根据Holder范数形式进行构造了失真函数。多方向的优点是减少了隐写分析算法从所用的方向进行建模检测到隐秘信息的可能性。在WOW中，先对载体进行镜像填充（Mirror-padded），之后在计算卷积，因此，𝑹(𝑘)的尺寸与原图 𝑿 相同。
