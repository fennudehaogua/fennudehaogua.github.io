HILL&MiPOD

发表于 2019-06-09

Hill

论文：A new cost function for spatial image steganography(2014,ICIP,CR20)

Investigation on cost assignment in spatial image steganography(2014,TIFS,CR12)

总结：HILL隐写的优点在于其代价函数的设计，从而使得嵌入的位置尽可能的在纹理复杂区域，因为之前介绍的隐写方法如WOW，S-UNIWARD等，它们的失真函数设计使得有的纹理复杂区域像素适合携带隐秘信息却赋有较高的嵌入损失，显然在这些纹理复杂的区域像素聚集嵌入信息比在光滑区域嵌入信息更为安全。

在加性失真模型下通过取倒数的方式及两个低通滤波器的配合使用，不仅实现了残差绝对值相对较大的区域(纹理复杂度较高)具有较低的代价值，从而具有较高的信息嵌入概率，而残差绝对值相对较小的区域(纹理复杂度较低)具有较高的代价值，从而具有了较低的信息嵌入概率，而且还实现了具有较低代价值的像素点相对比较聚集。这些具有高嵌入概率的像素，集中地分布在那些难以被检测的成块的、纹理复杂度较高的区域(例如噪声区、纹理区等)中。

在非加性失真模型下提出聚集隐写改变方向策略，结合该策略采取分批动态更新的方式设计出新的非加性代价函数。该价函数并不像加性代价函数一样，一次性获得所有像素的代价值，而是分批获得。同时，该非加性代价函数的设计分两阶段完成。第一阶段，借用己有的加性代价函数，但需要基于像素之间的相关性将部分信息的嵌入影响考虑入内。简而言之，这一阶段的代价值(初始代价值)，需要根据己携带有部分秘密信息的携密图像而不是载体图像获得。第二阶段，为结合聚集改变方向策略，根据像素己有的嵌入结果调整当前代价值。此时，对于加“1”和减“1”的代价值不再采取同一个代价函数获得，而是采取结合了聚集改变方向策略的不同代价函数获得。代价值的调整原理大致为，如果相邻像素发生的改变是加“1”(减“1”)则将当前像素加“1”(减“1”)的代价值适当降低，减“1”(加“1”)的代价值适当提高，使得当前像素如果需要发生改变，则尽可能地加“1”(减“1”)。

加性失真：

HILL验证了聚集修改策略在一定程度上可以提高隐写安全性能，设计了一种让载体中低失真值更加聚集的失真函数HILL（HIgh-pass,Low-pass, Low-pass)。即利用两个低通滤波器和一个高通滤波器设计失真函数，同时在嵌入秘密信息时引入了聚焦改变位置策略，取得了比之前几种自适应隐写算法更好的抗检测性能。

在WOW，S-UNIWARD等自适应隐写术中，它们都致力于将秘密信息嵌入到纹理复杂区域，然而，有一些在纹理复杂区域的适合携带秘密信息的像素仍被设定了较大的嵌入损失。相比起在光滑区域，在这些像素中嵌入秘密信息更加安全。

HILL算法的失真函数设计借助了一个高通滤波器和两个均值滤波器。HILL算法的构造方式与WOW类似，其利用KB预测算子对载体图像进行了分解获取图像残差，并提出一种聚焦改变位置策略，目的是在确保秘密信息可以全部被嵌入到载体中的前提下，尽量让秘密信息分布在纹理丰富区域，减小其被建模检测到的可能性。假设载体图像为X，则HILL算法的失真函数有式的表达形式：

非加性失真（HILL-CMD）：

由于直接基于非加性失真函数不易设计隐写算法，因此需采用可行的方式进行算法的设计。Filler尝试了采用加性失真函数形式或失真上界来实施非加性失真函数模型下隐写算法的设计。在HILL中，设计了一种结合了聚集改变方向策略的新代价函数，该代价函数在获取像素代价值时采用分批动态更新的方式。利用该代价函数构建出一类具有高安全性的图像空域隐写算法。由于该类算法捕获了信息嵌入改变之间的相互作用，得到的失真不是由所有像素的改变代价值相加而得，因此该类算法属于非加性隐写算法。

总结：
