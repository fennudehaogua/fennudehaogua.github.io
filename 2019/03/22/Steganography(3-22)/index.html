EA-LSBMR

论文：Edge adaptive image steganography based on LSB matching revisited(2010,TIFS,CR334)

 简单LSB替换算法虽然仅对载体图像进行微小的修改，但仍会因在最低位平面产生某些异常特性如值对现象而容易被卡方检测等统计方法识别出来。为了避免这种统计异常性的最简单方法就是LSB 匹配算法(LSB Matching algorithm,LSBM)：当嵌入的比特与像素值最低位相同时，像素值不变；不同时随机选择 +1或–1。在达到最大嵌入率时，LSB 替换、LSBM 的像素改变率均为 0.5比特/像素。文献[Mielikainen J. LSB Matching Revisited[J]. IEEE Signal ProcessingLetters, 2006, 13(5): 285-287. ]提出 LSB 匹配改进算法(LSB Matching Revisited algorithm,LSBMR)
 该算法将像素对作为一个嵌入单元进行秘密信息嵌入，像素对中的第一个像素的LSB 携带一个比特秘密信息，2个像素之间的函数关系(奇偶)携带另一比特秘密信息。在达到最大嵌入率时，LSBMR的像素改变率为 0.375 比特/像素。LSBMR 既能克服 LSB替换引起的不对称现象，在相同嵌入率条件下对载体的改动也较小，安全性也较高。
 多数隐写算法通常认为, 自然图像的LSB位平面是随机分布的。但这种假设并不是都成立的,特别是对于有很多平坦区域的图像与平坦区域相比, 位于边缘区域的像素LSB随机性强,且边缘区域与图像自身内容相关, 在这些区域进行信息的嵌入更加不容易被检测。


 一幅图像的不同区域拥有不同的承载秘密消息的能力，在同等安全性的前提下，纹理复杂区域较平坦区域能负载更多的秘密消息比特。为了在低嵌入率下充分利用图像中纹理复杂区域隐藏秘密消息，提出将LSBMR算法应用到图像边缘区域的像素对中，提出了一种新的边缘强度自适应隐写算法，称为EA-LSBMR隐写算法。

EA-LSBMR算法将图像分成两两一组的像素对，在给定的嵌入率下首先选取差分值较大的像素对嵌入消息，因此，在嵌入率较低时，相邻像素差分值较小的平坦区域就会不嵌或嵌入较少的信息;这样相邻像素差分值较大的纹理复杂区域自身统计特性的变化就可以掩盖消息嵌入带来的像素对相关性的改变，从而抵抗盲检测算法的攻击。

在介绍EA-LSBMR隐写算法之前，我们首先理解LSBMR，对于给定得以一个载体图像像素对（ai,a(i+1)），载密像素对数(Ai,A(i+1))，则秘密信息比特mi，m(i+1)依据以下四种方式嵌入：

  1. 如果LSB（ai)= mi，且f（ai,a(i+1)）= m(i+1)，则(Ai,A(i+1))= (ai,a(i+1))

  2. 如果LSB（ai)= mi，且f（ai,a(i+1)）≠ m(i+1)，则(Ai,A(i+1))= (ai,a(i+1)+r)

  3. 如果LSB（ai)≠mi，且f（ai -1,a(i+1)）= m(i+1)，则(Ai,A(i+1))= (ai-1,a(i+1))

  4. 如果LSB（ai)≠mi，且f（ai -1,a(i+1)）≠ m(i+1)，则(Ai,A(i+1))= (ai+1,a(i+1))

其中，LSB (ai))指像素值ai的最低比特位，函数f定义为f (a,b) = LSB(La/2˩+ b),r为-1和1中的一个随机值。
